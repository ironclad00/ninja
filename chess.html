Overview

Singularity chess is a chess variant where all the squares converge in the center of the board. I started working on a 2-player version (no AI) of this game as a challenge to myself to make a game/app using nothing but standard web technologies. It has proven to be more difficult than I thought since the warped board renders the meaning of standard directions like "forwards" and "backwards" virtually meaningless.

Right now the game is still being developed, but it should be almost 100% playable. Unfortunately, since Fall classes have now started, I may not have much time to work it, but I'll update it whenever I can.

Features & Bugs

- Feature: Clicking on a piece will highlight all spaces that said piece can move to

- Feature: Players cannot make moves that will put their king in check

- Feature: Checkmate detection and game over alert

- Feature: No known bugs (on Chrome)! I'm sure this will change soon...
