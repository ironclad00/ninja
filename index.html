About Me

Hello and welcome to my portfolio site! I'm Jeremy Baker, a 5th-year computer science undergraduate student at the University of Florida. I designed and created this site in the hope of providing an overview of my experience with projects whose scope and/or content is hard to convey with a single-page resume (especially anything related to graphic design and art). Feel free to message me about any of the content you see here! :)

My interests are quite varied across all types of disciplines. Concerning computer science, I enjoy UI/UX design more than any other aspect. The psychology behind what makes a design "good" or "bad" in human-computer interactions fascinates me.

I also enjoy mathematical puzzle solving using programming. I have recently discovered Project Euler and have started posting the source code for my solutions in the Euler tab.


About This Site

I made this site with Bootstrap 3. Boostrap is awesome and I highly recommend it for anyone interested in responsive web design. I also used two jQuery libraries called Transit and Raphael. The syntax highlighter I use for displaying code was created by Alex Gorbachev.

All of the code and files for this site are stored on my public GitHub repository ninja. Please Note: Since I do not own any apple products, I have not yet tested my site extensively on iPhones, iPads, or Safari in general. I will fix any bugs occuring on these devices as soon as I can.
